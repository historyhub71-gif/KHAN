# 🎓 Institute Management System

A modern **Institute Management System** built with **Expo (React Native)** and **Supabase**, designed to simplify educational institution management through secure role-based access, automated workflows, and real-time notifications.

# 🔄 Admission Workflow

```text
Admin Creates Admission Deal
            │
            ▼
Student Assigned to Interviewer (ASR)
            │
            ▼
ASR Conducts Assessment
            │
            ▼
Admin Reviews Assessment
            │
            ▼
Admin Approves Admission
            │
            ▼
Teacher Assigned
            │
            ▼
Student Can Register
```

---

# 📈 14-Day Progress Review Workflow

```text
Admission Approved
        │
        ▼
Review #1 (Day 15)
        │
        ▼
Review #2 (Day 30)
        │
        ▼
Review #3 (Day 45)
        │
        ▼
Review Cycle Completed
```

# ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/YOUR_REPOSITORY.git
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npx expo start
```
